# LogAnalyzer
Приложение парсит неструктурированные текстовые логи, фильтрует инциденты по ключевым словам и предоставляет контекст ошибок.

Всем привет, выкладываю свой проект, который когда-то разрабатывал будучи фрилансером. Он представляет из себя парсер огромных файлов, размер которых ограничен лишь вашим железом.
Программа не ест оперативку, все зависит от скорости вашего диска. Разрабатывалось приложение для администрирования серверов, которые записывают свое состояние в файл.
Как работает:
1. Вы вводите путь к файлу, нажимаете open (если кнопка загорелась зеленым - путь верный)
2. Вводите искомое слово, например ERROR, CRITICAL или другие состояния которые вам требуются, нажимаете find
3. Если программа нашла слово, она берет 2 строки до и 4 после этого слова и выводит на экран. Это сделано для того, чтобы админы могли найти причину ошибки, что ей предшествовало, выявить 
причину поломки, посмотреть как дальше работал сервер и.т.д.
4. Вы можете листать такие контексты назад или вперед с помощью кнопок next и prev. Выход из программы - close.
